# Sparse-Matrix-Linked-List-Representation
Sparse Matrix Linked List Representation

if Matrix is included most of elements zero and we need to only non-zero elements, we can use sparse matrix

For example we use this matrix in code.

|   3   5   0   | <br>
|   0   0   2   | <br>
|   0   4   0   | <br>
|   2   0   3   |
